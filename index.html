OMNICheck Installation Guide

This wizard will guide you through the installation of PostgreSQL, pgAdmin, and ODBC for OMNICheck.

OMNICheck Installation Wizard

Hardware Step 1: Unbox & Connect

Connect power, display, keyboard/mouse, and network cable to the OMNICheck machine.

Hardware Step 2: Power On & BIOS Check

Verify boot order (SSD first) and enable virtualization if required.

Step 1: Run Installer as Administrator

Right-click the installer file and select Run as administrator to begin installation.

Step 2: Destination Directory

Choose the folder where the application will be installed, then click Next.

Step 3: Start Installation

Review the installation summary. Click Next to begin installation.

Step 4: Installation Complete

The installer confirms once installation is complete, Please Click Next to Proceed with Database Installation .
